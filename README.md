# ViralSim - Simulador de Viralidad

Simula la viralización de una red social con Procesos de Poisson y Cadenas de Markov

## Instalación

Sigue estos pasos para instalar y ejecutar el proyecto en tu máquina local:

1. **Clona el repositorio:**
```bash
   git clone https://github.com/IvarP3/ViralSim---Simulador-de-Viralidad.git
```

2. **Navega al directorio:**
```bash
   cd nombre_o_direccion_del_repositorio
```

3. **Crea un entorno virtual** (si no quieres instalar dependencias de forma local):
```bash
   python -m venv venv
```

4. **Activa el entorno:**
```powershell
   .\venv\Scripts\Activate.ps1
```

5. **Instala las dependencias:**
```bash
   pip install -r requirements.txt
```

## Uso

Instrucciones sobre cómo ejecutar la aplicación:
